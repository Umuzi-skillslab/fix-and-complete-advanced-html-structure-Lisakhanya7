# Website Wireframe and Layout Strategy

## Site Structure Overview

### Navigation Flow
```

├── Header/Navigation
├── Hero Section

About Page (about.html)
├── Header/Navigation
├── Team Information
├── Team Members (3 profiles)
├── Services Breakdown (3 cards flexbox)

Media Page (media.html)
├── Header/Navigation
├── Video Showcase (2 videos with controls)
├── Audio Sample (audio player)
├── Location with Embedded Map (iframe)

Contact Page (contact.html)
├── Header/Navigation
├── Contact Information
├── Comprehensive Contact Form (6 fieldsets)
│   ├── Personal Information (4 inputs)
│   ├── Project Information (4 inputs)
│   ├── Services Interest (5 checkboxes)
│   ├── Experience Level (4 radio buttons)
│   ├── Project Details (textarea)
│   └── Additional Info (2 selects, 2 checkboxes)
├── FAQ Section
└── Footer
```

## Responsive Breakpoints

### Desktop (1200px+)
- Full width content with max-width container
- Navigation items displayed horizontally
- Services cards: 3 columns (flex)
- Portfolio gallery: 3-4 columns (grid)
- Hero height: 500px
- All flexbox and grid features active

### Tablet (768px - 1199px)
- Slightly reduced content width with padding
- Navigation items still horizontal with wrapping enabled
- Services cards: 2 columns on medium tablet, 3 small tablets
- Portfolio gallery: 2-3 columns
- Hero height: 400px
- Grid and flexbox responsive features engaged

### Mobile (480px - 767px)
- Full width with mobile padding
- Navigation stacks vertically
- All cards stack to 1 column
- Portfolio gallery: single column
- Hero height: 300px
- Simplified layout, all functionality preserved

### Small Mobile (<480px)
- Full width, optimized padding
- Navigation: vertical stack
- Single column everything
- Hero height: 250px
- Touch-friendly buttons (larger tap targets)

## CSS Features by Layout Type

### Flexbox Features Used
1. Navigation: `flex-direction: row` → `column` (mobile)
2. Services: `justify-content: space-between`, `align-items: stretch`
3. Responsive wrapping with `flex-wrap: wrap`

### Grid Features Used
1. Portfolio: `grid-template-columns: repeat(auto-fit, minmax(280px, 1fr))`
2. Auto-fit responsive columns
3. Grid-gap for spacing

### Animation Features
1. Hero fade-in on page load
2. Slide-in animations for content
3. Hover scale effects on cards
4. Smooth transitions on all interactive elements

## Design System

### Color Palette
- Primary: #1a1a1a (Dark)
- Secondary: #ff6b35 (Orange)
- Accent: #004e89 (Blue)
- Light Background: #f5f5f5
- Text Dark: #333
- Text Light: #666
- White: #fff

### Typography
- Font Family: 'Segoe UI', Tahoma, Geneva, sans-serif
- h1: 2.5rem, gradient text
- h2: 2rem, text-shadow
- h3: 1.5rem
- p: 1rem, line-height 1.6

### Spacing
- Gap/margin: 2rem (sections), 1rem (inner)
- Padding: 2rem (cards), 1rem (text)

### Effects
- Transitions: 0.3s ease
- Transforms: scale, translate, rotate
- Animations: 2s, infinite loop on pulse

## Consistency across Pages

All pages maintain:
- Same header/navigation structure
- Consistent footer design
- Same color scheme
- Responsive behavior at all breakpoints
- Accessible forms and interactive elements
- Proper semantic HTML structure
